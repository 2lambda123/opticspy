opticspy

Opticspy is a python module for optics application. I want to this for a long time. One reason is I know both optics and python, so why no develop some optics tools? The second reason is that there is not much opensource, easy-to-use optics program module(Matlab has great functions but do not specify to optics application). I want this module could be developed in to a core for a future web application for optics. Now it is concentrated on interferometers, optical metrology and wave propagation. And most methods in opticspy based on Zernike Polynomials.

There are some examples opticspy module:

1. Zernike Polynomial Methods

2. Zernike Polynomial Fitting Methods

3. Interferometer Methods

4. Phase shift interferometer and surface rebuild

5. Wave Propagation and Aberration Methods

This project is hosted on github in https://github.com/Sterncat/opticspy

How to use

>>> import opticspy

And just have fun with it!

What I want

  • 1. After import the module and you will get some functions that can do some calculation and education in optics
  • 2. Parameters should be very flexible, and the results should be shown in visualized, intuitionistic figures.
  • 3. 1. Based on Zernike Polynomials.
  • 4. After development of core, I want it be dynamic web application(with Javascript)
  • 5. I even hope one day it will become a part of online optics design application, who knows?
